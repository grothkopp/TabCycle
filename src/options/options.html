TabCycle Settings

Time Mode

Choose how tab age is measured.

Thresholds

Set when tabs transition between statuses. Values must be in ascending order.

Group Age Display

Show the age of each tab group in its title (e.g. "News (23m)").

Bookmark Closed Tabs

Save tabs and tab groups closed by TabCycle as bookmarks for later recovery.

When enabled, tabs that reach the "Gone" state are saved as bookmarks in a dedicated folder under Other Bookmarks.
